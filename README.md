# Online Store Sales Analysis using SQL

## Project Overview
This project focuses on analyzing online store sales data using SQL to extract meaningful business insights related to sales performance and customer behavior.

## Objectives
- Analyze sales and order data using SQL
- Identify revenue trends and key performance metrics
- Understand customer purchasing patterns
- Support data-driven business decisions

## Tools & Technologies
- SQL
- Relational Database Concepts

## Key Analysis Performed
- Data filtering using WHERE conditions
- Aggregations using SUM, COUNT, and AVG
- Grouping data using GROUP BY
- Sorting and limiting results for insights
- Business-focused queries for sales analysis
